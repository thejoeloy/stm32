# SPH0645 I2S Mic

## Pin Connections

## Blue Push Button
Blue Push Button (PC13)

## SPH0645
3.3 -> 3.3V
GND -> GND
BCLK -> PB10 (I2S2_CK)
DOUT -> PC1 (I2S2_SD)
LRCL -> PB12 (I2S2 WS)
DEL -> None


### SD Card
CMD -> PD2 (SDIO_CMD)
CK -> PB2 (SDIO_CK)
D0 -> PC8 (SDIO_0)
CD -> PC6 (GPIO_Input / SD_Detect)
